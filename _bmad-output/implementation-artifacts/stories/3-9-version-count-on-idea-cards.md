# Story 3.9: Version Count on Idea Cards

**Epic:** Epic 3 - Idea Management & Dashboard
**Story Key:** 3-9-version-count-on-idea-cards
**Status:** backlog
**Acceptance Criteria Count:** 1
**Dependencies:** Stories 3.2, 3.4

---

## Story

As an **user viewing my ideas**,
I want **to see how many versions each idea has**,
So that **I can see which ideas I've iterated on**.

---

## Acceptance Criteria

### AC 1: Version Count Displayed

**Given** I view idea cards or list
**When** Each idea is displayed
**Then** I see version count (e.g., "v1", "v3", "v5") next to or in the card

---

## Tasks

### Frontend Tasks
1. Add version count to idea cards
2. Add version count to list items
3. Display version information

### Testing Tasks
1. Test version count displays
2. Test correct version shown

---

## Definition of Done

- [ ] Version count added to idea cards
- [ ] Version count added to list items
- [ ] Correct versions displayed
- [ ] All ACs passing
- [ ] Code committed
