# Story 3.11: Last Edited Timestamp on Idea Cards

**Epic:** Epic 3 - Idea Management & Dashboard
**Story Key:** 3-11-last-edited-timestamp-on-idea-cards
**Status:** backlog
**Acceptance Criteria Count:** 1
**Dependencies:** Stories 3.2, 3.4

---

## Story

As an **user managing my ideas**,
I want **to see when each idea was last edited**,
So that **I can prioritize working on ideas that haven't been updated recently**.

---

## Acceptance Criteria

### AC 1: Last Edited Timestamp Shows

**Given** I view idea cards or list
**When** Each idea is displayed
**Then** I see last edited date in human-readable format (e.g., "Edited 2 days ago" or "Today at 3:45 PM")

---

## Tasks

### Frontend Tasks
1. Add last edited timestamp to idea cards
2. Add to list items
3. Format timestamp for readability (relative or absolute)

### Testing Tasks
1. Test timestamp displays
2. Test format is readable

---

## Definition of Done

- [ ] Last edited timestamp added to cards
- [ ] Timestamp added to list items
- [ ] Format readable and correct
- [ ] All ACs passing
- [ ] Code committed
